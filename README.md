# 📱 Social Media AI Agent (Mini Project)

This is a **mini AI project** that generates social media posts using AI and includes a **human-in-the-loop (HITL) approval system**.

## 🚀 Features

* Generate social media posts based on a topic
* Multiple writing styles (Casual, Professional, Funny, Informative)
* AI-based content review for safety and quality
* Human approval before saving posts
* Saves approved posts with timestamp

## 🛠️ Tech Stack

* Python
* Google Gemini API
* dotenv

## ⚙️ How it works

1. User enters a topic
2. AI generates a post
3. System reviews content for safety
4. User approves/rejects the post
5. Approved posts are saved

## ▶️ Run the project

```bash
python sm_agent.py
```
